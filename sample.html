SVG Progress Ring Widget

An SVG is simply generated within the target tag using JavaScript. Technically, no CSS is required to draw the ring itself.

Usage

Create your target tag

  • Set an ID on a tag (usually a div) on where you want the ring to appear. You'll also likely want to set a width and height on this tag. If not, the ring will take up the full width of the tag.

Set arguments

  • Create an object in JS with arguments (see Arguments section below for reference)

Method 1 - Create as static object

  • If you don't plan on manipulating the ring further, simply call the function createProgressRing(args); where args are the desired arguments to pass to it.

Method 2 - Create as class object

  • If you do want manipulate the ring, you can set a variable and call the method new progressRing(args); where args are the desired arguments to pass to it.

Arguments

Required Arguments

  • target (string): The ID of the tag you want to put a ring inside of
  • progressPercent (float): What percent to show? (needs to be represented as a number from 0 to 1)

Optional Arguments

  • radius (float): Size of the radius (defaults to 100)
  • setParentWidth (bool): If set to true, will set the width of the containing tag to that of the radius. If false, the SVG will take up the entire width. (defaults to false)
  • progressWidth (float): Width of the progress bar (defaults to 25)
  • progressBorderWidth (float): Border width of the progress bar (set to 0 for none; defaults to 1)
    â€»progressWidth and progressBorderWidth cannot be combined to a total greater or equal to the radius!
  • progressBGcolor (string): Color of the area behind the progress bar
  • progressColor (string): Color of the progress bar
  • progressBorderColor (string): Color of the border of the progress bar (set to "none" for none)
  • centerColor (string): Color of the center (set to "none" for none)
  • thresholds (array of objects): Optionally have the progress color change upon dropping below certain levels. Keys are:
    level (float): The percent level to check. (number between 0-1)
    color (string): What color to change to upon dropping below this level

Methods

  • setProgress(float): Set the percent of progress
